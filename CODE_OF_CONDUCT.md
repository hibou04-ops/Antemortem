# Code of Conduct

This project adopts the [Contributor Covenant v2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/) as its code of conduct.

## Reporting

Issues that warrant a code-of-conduct report can be raised via a GitHub issue titled `Conduct: <short description>`. For private concerns, open an issue titled `Conduct: private contact request` and a maintainer will reach out through the address on the reporter's GitHub profile.

## Enforcement scope

This code applies to all project spaces — issues, pull requests, discussions — and to public representations of the project where the reporter can be reasonably identified with it.
